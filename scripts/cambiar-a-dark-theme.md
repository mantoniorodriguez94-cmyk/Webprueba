# Cambios de White Theme a Dark Theme

## Reemplazos Necesarios:

### Fondos
- `bg-white` → `bg-gray-800`
- `bg-white/90` → `bg-gray-800/90`
- `bg-white/85` → `bg-gray-800/90`
- `bg-white/80` → `bg-gray-800/80`
- `bg-white/70` → `bg-gray-800/70`
- `bg-white/60` → `bg-gray-800/60`
- `bg-white/50` → `bg-gray-800/50`
- `from-[#E3F2FD]` → `from-gray-700`
- `to-[#BBDEFB]` → `to-gray-600`

### Textos
- `text-gray-900` → `text-white`
- `text-gray-800` → `text-gray-100`
- `text-gray-700` → `text-gray-200`
- `text-gray-600` → `text-gray-300`
- `text-[#0288D1]` → `text-blue-400`

### Bordes
- `border-white` → `border-gray-600`
- `border-white/40` → `border-gray-600/40`
- `border-white/50` → `border-gray-700/50`
- `border-[#0288D1]/20` → `border-blue-500/20`
- `ring-white` → `ring-gray-700`

### Otros
- `shadow-lg` → `shadow-2xl` (para mejor visibilidad en dark)
- Mantener los gradientes de colores (blue, purple, green, etc.)
